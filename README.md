# Data Projects Portfolio

This repository contains four data analysis projects I completed:

1. **Football Institution Players and Coaches (SQL)**  
   - Creation of multiple tables with all players and coaches across categories.  
   - SQL queries for data retrieval and analysis.

2. **SQL Problems from Data Analyst Course**  
   - Exercises and solutions using SQL.  
   - Focus on joins, aggregations, and advanced queries.

3. **World Cup Data Analysis (Python - Pandas)**  
   - Analysis of matches from all World Cups using Python and Pandas.  
   - Includes data cleaning, calculations, and visualizations.

4. **New York Parking Violations Analysis (Power BI)**
